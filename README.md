## 프로젝트 소개

**1인가구**가 증가하는 사회에서 집에 남아있는 **반려동물과의 소통, 안전**을 위한 **스마트홈 서비스**

### 목적

- 반려동물이 증가하지만 함께 보내는 시간이 부족한 사회에서 실시간 영상을 통해 반려동물을 볼 수 있고 알맞은 환경을 조정할 수 있다. 
- 바쁜 사회에서 원격으로 집의 가전제품을 제어하여 편리함을 준다.
- 실시간 영상을 이용하여 방범 CCTV역할을 한다.
